Distribution:

/docs: javadocs for both the core and harness frameworks

/mashlib: dependent libraries needed for running core mash
/junitrunnerlib: dependent libraries needed for running Junit runner
/dbharnesslib: dependent libraries needed for running db harness
/httpharnesslib: dependent libraries needed for running http harness

/src: all the source for the packaged modules

/mash.jar: the mash core
/JunitRunner.jar: the junit functionality (good for integrating into ant)
/DBHarness.jar: the db interaction harness
/HttpHarness.jar: the http interaction harness
